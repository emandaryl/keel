# Keel

**Turn an idea into a finished app — no coding required.**

Keel is a free kit you open with an AI coding assistant (Claude Code, Cursor, or
Antigravity). You describe what you want in plain words; the assistant plans it
with you, builds it, checks that it works, and hands you a finished app.

- New here? Open **START-HERE.md**.
- Want the full walkthrough? Open **QUICKSTART.md**.

Free to use — personal or commercial. No sign-up.
